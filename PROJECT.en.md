# About This Project

This repository is a **personal practice** project built on top of [Vue Vben Admin](https://github.com/vbenjs/vue-vben-admin) via **secondary development**. It is intended for learning and experimenting with frontend engineering practices and admin-dashboard–style applications.

## Upstream Open Source Project

The original / upstream repository:

- **https://github.com/vbenjs/vue-vben-admin.git**

Implementation ideas, components, and documentation here are largely informed by the upstream project. Changes in this fork may include experimental edits, sample business logic, and custom configuration, and **do not** represent an official release.

## Notes

- The work is primarily for practice and experimentation; code and configuration are **not** guaranteed to be production-ready.
- For official documentation, updates, and community support, refer to [vue-vben-admin](https://github.com/vbenjs/vue-vben-admin).
- Please comply with the upstream project’s license and terms.
